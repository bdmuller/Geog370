In this project I learned how to display images in different projections

How the maps were made:

The maps were created using QGIS. Within QGIS, I used the globe editor as well as the indicatrix mapping tool
to modify the different map projections of the base layer.

Gall-Peters Projection

The Gall-Peters Map is categorized as a cylindrical projection map. This map displays the countries
in proper proportions to one another. A drawback to this map is that there is severe distortion at the poles.
The map was presented by James Gall in 1973, and mimics an earlier map created by Arno Peters almost a century
earlier.

Winkel Tripel Azimuthal

The Winkel-Tripel Map is categorized as a azimuthal projection map. This map is especially good for observing
climate, vegetation, and soil. This is because of the relatively small amount of distortion regarding the areas, distance
and direction of continents, countries, and islands. This map was named after its creator, Oswald Winkel in 1921
and accepted by the National Geographic Society in 1998.

North Pole Azimuthal Equidistant

Known for preserving both distance and direction, the North Pole Azimuthal Equidistant map is most used for polar routes
for navigating both air and sea.

World Equidistant Conical

Based on the archaic projection map of Ptolemy (A.D. 100), the World Equidistant Conical Map is best used for
viewing land masses extending in an East/West direction. Examples for using this type of mapping would
include mapping Russia.

World Cylindrical Equal Area

Created in 1772 by Johann H. Lambert, the cylindrical equal area map is mainly used for large-scale
mapping projects for the areas near the equator and wouldn’t be recommended for smaller scale “world” maps.

Sphere Winkel 1

This map is generally not used because it distorts shapes, areas, distances, directions, and angles.
This style of map can only be used on spheres as the semimajor axis would be used on ellipsoids.

Pseudo-Mercator

Otherwise known as the standard for mapping, the Mercator Map is a cylindrical projection. While any
straight line drawn on this style of map will indicate precise compass headings, it doesn’t represent the
shortest distance between two points. This map was originally created in 1569. The variant “Web Mercator”, is
still used in most web mapping and online services.
